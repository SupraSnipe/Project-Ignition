# Project Ignition Roadmap

> **Vision**
>
> Project Ignition is a self-hosted Workshop Operating System that helps makers, engineers, hobbyists, and small manufacturers move from ideas to completed projects with as little friction as possible.

---

# Current Status

**Current Version:** `0.7.0-alpha`

**Current Milestone:** Milestone 1 – Planning & Execution

**Current Focus:** Planning Experience.

---

# Development Philosophy

Every feature should answer one question:

> **Does this reduce friction between an idea and making progress?**

If the answer is "no", the feature should be reconsidered.

# Version 1.0 Success Criteria

Version 1.0 is complete when a user can:

* Manage multiple projects from beginning to completion.
* Plan daily work from the dashboard.
* Receive intelligent task recommendations from the Ignition Engine.
* Track project progress.
* Manage inventory and BOMs.
* Reliably back up and restore data.
* Deploy the application to a PC or NAS using Docker in minutes.

---

# Long-Term Vision

Project Ignition should become a modular Workshop Operating System that builders genuinely enjoy opening every day—not simply because it stores information, but because it helps them make meaningful progress on the projects they care about.
