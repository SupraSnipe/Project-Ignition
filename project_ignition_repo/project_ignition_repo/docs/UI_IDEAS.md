# UI Ideas

## Dashboard

The dashboard should answer:

> What should I work on right now?

Possible sections:

- Today's Focus
- Due This Week
- Overdue Tasks
- Quick Add
- Project Health
- Recently Updated
- Current Build Session

## Ignition Button

Future feature: select one to three useful tasks based on due date, priority, dependencies, estimated time, and current focus.
